# Project Configuration Preferences

## Version Management

**Node.js and TypeScript versions should NOT be modified unless explicitly requested by the user.**

- Do not update `package.json` to change Node.js version requirements
- Do not modify TypeScript version in `package.json` dependencies or devDependencies
- Do not suggest or apply version upgrades for Node.js or TypeScript
- Respect the existing version constraints in the project

When suggesting changes to dependencies or project configuration, preserve the current Node.js and TypeScript versions unless the user specifically asks to upgrade or modify them.
